# نصب Cafe POS روی ویندوز

این برنامه روی یک کامپیوتر ویندوزی و به صورت محلی اجرا می‌شود:

| بخش | آدرس |
| --- | --- |
| POS | http://localhost:3001 |
| پنل مدیریت | http://localhost:3002 |
| API | http://localhost:3000 |

## پیش‌نیازها

1. Node.js LTS نسخه 20 یا 22 را نصب کنید.
2. بعد از نصب Node، در PowerShell یا CMD اجرا کنید:

```bat
corepack enable
```

## نصب و آماده‌سازی

در ویندوز، وارد پوشه پروژه شوید و این فایل را اجرا کنید:

```bat
scripts\win\Setup.cmd
```

این دستور وابستگی‌ها را نصب می‌کند، فایل‌های تنظیمات محلی را می‌سازد، برنامه را build می‌کند، دیتابیس SQLite را می‌سازد و داده‌های اولیه را وارد می‌کند.

## Step 5 — Verify Installation

قبل از اجرای برنامه، این فایل را اجرا کنید:

```bat
scripts\win\Check.cmd
```

این بررسی نشان می‌دهد نصب برای اجرا آماده است یا نه:

- `✔ PASS` یعنی آن بخش درست است.
- `⚠ WARNING` یعنی نصب هنوز کامل نیست یا یک مرحله مثل ساخت دیتابیس انجام نشده است.
- `✖ FAIL` یعنی یک پیش‌نیاز یا خروجی لازم وجود ندارد و باید خطا را رفع کنید.

فقط وقتی انتهای خروجی `READY FOR STARTUP` باشد، برنامه را با `Start.cmd` اجرا کنید. اگر `SETUP INCOMPLETE` دیدید، ابتدا `Setup.cmd` را دوباره اجرا کنید و سپس `Check.cmd` را تکرار کنید.

## اجرای برنامه

بعد از setup، این فایل را اجرا کنید:

```bat
scripts\win\Start.cmd
```

پنجره POS به صورت خودکار باز می‌شود. پنل مدیریت هم از این آدرس در دسترس است:

```text
http://localhost:3002
```

برای توقف برنامه:

```bat
scripts\win\Stop.cmd
```

## میانبر دسکتاپ

برای اجرای بدون پنجره CMD، در ویندوز یک shortcut به این فایل بسازید:

```bat
scripts\win\CafePOS.vbs
```

## فایل دیتابیس و بکاپ

دیتابیس این نسخه در این مسیر ساخته می‌شود:

```text
apps\backend\data\cafe.db
```

برای بکاپ گرفتن، همین فایل را کپی کنید. برای انتقال اطلاعات به سیستم دیگر، برنامه را ببندید و فایل `cafe.db` را به همان مسیر در سیستم مقصد منتقل کنید.

## خطاهای رایج

- اگر پورت‌های 3000، 3001 یا 3002 اشغال بودند، `scripts\win\Stop.cmd` را اجرا کنید و دوباره Start بزنید.
- اگر `corepack` یا `pnpm` شناخته نشد، CMD/PowerShell را ببندید و دوباره باز کنید. اگر درست نشد، Node.js LTS را دوباره نصب کنید و `corepack enable` را اجرا کنید.
- اگر پنل مدیریت خطای API داد، یک بار `scripts\win\Setup.cmd` را دوباره اجرا کنید تا فایل `apps\admin\.env.local` ساخته شود.
